# SkillBoxNode_HW2
##Домашнее задание для блока №3

Создан новый модуль date.js.
Он  экспортирует функцию с именем currentDateTime, она не получает никаких аргументов и возвращает объект вида 
{ date: '2020-08-05', time: '12:13:14' }.

Из главного модуля index.js сделан импорт этой функции, при выполнении этот скрипт выводит строчку в формате: 
Today is 2020-08-05, the current time is 12:13:14.
